[10;1H[1mDESCRIPTION[0m[m
[11;1H       This manual page documents the GNU version of [1mchown[22m.  [1mchown [22mchanges the[m
[12;1H       user  and/or  group  ownership of each given file.  If only an owner (a[m
[13;1H       user name or numeric user ID) is given, that user is made the owner  of[m
[14;1H       each  given file, and the files' group is not changed.  If the owner is[m
[15;1H       followed by a colon and a group name (or numeric  group  ID),  with  no[m
[16;1H       spaces  between  them,  the  group ownership of the files is changed as[m
[17;1H       well.  If a colon but no group name follows the user name, that user is[m
[18;1H       made the owner of the files and the group of the files  is  changed  to[m
[19;1H       that  user's  login  group.   If the colon and group are given, but the[m
[20;1H       owner is omitted, only the group of the files is changed; in this case,[m
       [1mchown [22mperforms the same function as [1mchgrp[22m.  If only a colon  is  given,[m
       or  if  the entire operand is empty, neither the owner nor the group is[m
       changed.[m
