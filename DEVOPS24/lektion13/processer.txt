ps - listar processer

    ps -e
        Se alla
    ps -ejH
        -e     Select all processes.  Identical to -A.
    ps -eO pid,pri,pgrp,user
        Alla processer, med pid, prioritet, pgrp, user

kill -9 PID
    Stäng ned processen med id PID utan att låta den göra något

nice echo hej
    Sätt prioriten på echo lägre
