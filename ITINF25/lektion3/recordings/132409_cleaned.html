DESCRIPTION
       Tree is a recursive directory listing  program  that
       produces a depth indented listing of files, which is
       colorized ala dircolors if the LS_COLORS environment
       variable is set and output is to tty.  With no argu‐
       ments,  tree  lists  the files in the current direc‐
       tory.  When  directory  arguments  are  given,  tree
       lists  all the files and/or directories found in the
       given directories each in turn.  Upon completion  of
       listing  all  files/directories  found, tree returns
       the total number of files and/or directories listed.

       By default, when a symbolic link is encountered, the
       path that the symbolic link refers to is printed af‐
       ter the name of the link in the format:

           name -> real-path

       If the `-l' option is given and  the  symbolic  link
       refers to an actual directory, then tree will follow
       the  path  of the symbolic link as if it were a real
       directory.
